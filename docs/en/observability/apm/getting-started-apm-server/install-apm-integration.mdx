<DocAccordion buttonContent="**If you have an internet connection**" initialIsOpen>

An internet connection is required to install the APM integration via the ((fleet)) UI in ((kib)).

{/* lint ignore elastic-agent */}
1. Open ((kib)) and select **Add integrations** > **Elastic APM**.
1. Click **APM integration**.
1. Click **Add Elastic APM**.
1. Click **Save and continue**.
1. Click **Add Elastic Agent later**. You do not need to run an ((agent)) to complete the setup.

</DocAccordion>
