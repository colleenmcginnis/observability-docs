
Built-in data filters allow you to filter or turn off ingestion of the following types of data:

| Data type | Common sensitive data |
|---|---|
| <DocLink id="enObservabilityApmFiltering" section="http-headers">HTTP headers</DocLink> | Passwords, credit card numbers, authorization, etc. |
| <DocLink id="enObservabilityApmFiltering" section="http-bodies">HTTP bodies</DocLink> | Passwords, credit card numbers, etc. |
| <DocLink id="enObservabilityApmFiltering" section="personal-data">Personal data</DocLink> | Client IP address and user agent. |
| <DocLink id="enObservabilityApmFiltering" section="real-user-monitoring-data">Real user monitoring data</DocLink> | URLs visited, click events, user browser errors, resources used, etc. |
| <DocLink id="enObservabilityApmFiltering" section="database-statements">Database statements</DocLink> | Sensitive user or business information |
