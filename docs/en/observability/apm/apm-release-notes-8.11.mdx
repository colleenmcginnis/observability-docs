### Breaking Changes
- The `ecs.version` field has been removed from APM data streams [11632]({pull}11632)

<div id="bug_fixes"></div>
### Bug fixes
- Add back gzip support for grpc otlp endpoint [11434]({pull}11434)
- Correctly mark jvm.memory.non_heap.pool.* and jvm.fd.* metrics as internal [11303]({pull}11303)
- Fix tail-based sampling discarding low throughput and low sample rate traces [11642]({pull}11642)
- Add memory based autoscaling for service destination aggregation groups [11739]({pull}11739)

<div id="added"></div>

### Added
- Support and define DLM data retention period in the apmpackage [11539]({pull}11539)
- Expose new metrics into the local batch processor [11582]({pull}11582)
- Add geoip processing to app_logs ingest pipeline on `client.ip` [11699]({pull}11699)
