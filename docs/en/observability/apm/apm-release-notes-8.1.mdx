## APM version 8.1.3

[View commits](https://github.com/elastic/apm-server/compare/v8.1.2\...v8.1.3)

This release includes an important APM Server bug fix.
Users running APM Server version 8.0.0 and above should upgrade immediately.

<div id="bug_fixes"></div>

### Bug fixes
- Fix a bug that caused the APM Server to run out of memory when it receives events from a high number of APM agents [7809]({pull}7809)
