This upgrade guide is for the standalone method of running APM Server.
Only use this guide if both of the following are true:

* You're using ((ecloud)).
* You're using the APM Server binary, i.e. you haven't switched to the Elastic APM integration.

Follow these steps to upgrade:

1. Review the APM <DocLink id="enObservabilityApmReleaseNotes">release notes</DocLink>, <DocLink id="enObservabilityApmBreaking">breaking changes</DocLink>,
    and ((observability)) [What's new](((observability-guide))/whats-new.html) content.

1. Upgrade ((ecloud)) to ((version)),
    See [Upgrade versions](((cloud))/ec-upgrade-deployment.html) for instructions.

1. (Optional) Upgrade to the APM integration.
    Got time for one more upgrade?
    See <DocLink id="enObservabilityApmUpgradeToApmIntegration">Switch to the Elastic APM integration</DocLink>.
