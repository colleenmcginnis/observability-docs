## Configure APM

Configure APM by editing the `apm-server.yml` configuration file.
The location of this file varies by platform--see the <DocLink id="enObservabilityApmDirectoryLayout">Installation layout</DocLink> for help locating it.

A minimal configuration file might look like this:

```yaml
apm-server:
  host: "localhost:8200"  [^1]
output.elasticsearch:
  hosts: ["localhost:9200"]  [^2]
  username: "elastic"  [^3]
  password: "changeme"
```
[^1]: The `host:port` APM Server listens on.
[^2]: The ((es)) `host:port` to connect to.
[^3]: This example uses basic authentication.
The user provided here needs the privileges required to publish events to ((es)).
To create a dedicated user for this role, see <DocLink id="enObservabilityApmPrivilegesToPublishEvents">Grant privileges and roles needed for writing events</DocLink>.

All available configuration options are outlined in
[configuring APM Server](((observability-guide))/apm-configuring-howto-apm-server.html).
