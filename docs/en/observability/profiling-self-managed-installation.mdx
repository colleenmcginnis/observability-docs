1. <DocLink slug="/en/observability/profiling-self-managed-setup-kibana">Set up Universal Profiling in Kibana</DocLink>.
1. <DocLink slug="/en/observability/profiling-self-managed-running">Run the backend applications</DocLink>.
1. <DocLink slug="/en/observability/profiling-self-managed-install-next-steps">Next steps</DocLink>.

If you face any issues during installation, refer to <DocLink slug="/en/observability/profiling-self-managed-troubleshooting">Troubleshooting Universal Profiling backend</DocLink>.

After the Universal Profiling installation is complete, refer to <DocLink slug="/en/observability/profiling-self-managed-ops">Operating the Universal Profiling backend</DocLink> for more on monitoring and scaling the backend.
