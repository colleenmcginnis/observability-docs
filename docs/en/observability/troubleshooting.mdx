## Tools

Use the following tools in ((kib)) to troubleshoot issues across ((observability)) apps:

* <DocLink id="enObservabilityExploratoryDataVisualizations">Explore data</DocLink>: Select and filter result data in any dimension, and look
    for the cause or impact of performance problems.

* <DocLink id="enObservabilityInspect">Inspect</DocLink>: View information about all requests that were made to collect the data displayed on the current page.
