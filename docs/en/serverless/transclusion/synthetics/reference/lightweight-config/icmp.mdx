|===
| Option (type) | Description

| <span id="monitor-icmp-hosts">**`hosts`**</span><br />
(<<synthetics-lightweight-string,string>>)
a| **Required**. The host to ping.

**Example**:

[source,yaml]
----
hosts: "myhost"
----

| <span id="monitor-icmp-wait">**`wait`**</span><br />
(<<synthetics-lightweight-duration,duration>>)
a| The duration to wait before emitting another ICMP Echo Request if no response is received.

**Default**: `1s`

**Example**:

[source,yaml]
----
wait: 1m
----
|===
