=== HTTP 400: Data decoding error / Data validation error

The most likely cause for this error is using an incompatible version of an {apm-agent}.
See <<apm-agents-elastic-apm-agents-minimum-supported-versions,minimum supported APM agent versions>> to verify compatibility.

[[event-too-large]]

[discrete]
[[http-400-event-too-large]]
=== HTTP 400: Event too large

APM agents communicate with the Managed intake service by sending events in an HTTP request. Each event is sent as its own line in the HTTP request body. If events are too large, you can reduce the size of the events that your APM agents send by: <<apm-compress-spans,enabling span compression>> or <<apm-stacktrace-collection,reducing collected stack trace information>>.
